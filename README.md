# Neovim My Nice Configuration
```
F9	            Normal/Insert   	Spustí příkaz :make (pro kompilaci C/C++ projektů)
Ctrl + c	        Visual	        Zkopíruje označený text do systémové schránky (jako v OS)
Ctrl + h	        Normal	        Skok do okna vlevo
Ctrl + j	        Normal	        Skok do okna dolů
Ctrl + k	        Normal	        Skok do okna nahoru
Ctrl + l	        Normal	        Skok do okna vpravo
Mezerník + e	    Nvim-Tree	    Otevře/Zavře postranní strom projektu (Explorer)
- (Mínus)	        Oil.nvim	    Otevře aktuální složku pro editaci (jako text)
Shift + l	        Bufferline	    Přepne na další záložku (vpravo)
Shift + h	        Bufferline	    Přepne na předchozí záložku (vlevo)
Mezerník + c	    Bufferline	    Zavře aktuální záložku (buffer)
Mezerník + f + f	Find Files      Najít soubor podle názvu
Mezerník + f + g	Find Grep       Hledat text uvnitř všech souborů v projektu
Mezerník + f + b	Find Buffers    Najít otevřený soubor (záložku)
Mezerník + f + h	Find Help       Hledat v nápovědě Neovimu
Mezerník + f + r	Find Registers  Zobrazit obsah schránky/registrů
Mezerník + f	    Zformátuje kód (např. zarovná Python/Lua/C)
g + d	            Go Definition   Skočí na definici funkce/proměnné pod kurzorem
K	                Zobrazí dokumentaci k funkci pod kurzorem (v plovoucím okně)
Mezerník + r + n	Rename – Přejmenuje proměnnou pod kurzorem v celém projektu
Mezerník + c + a	Code Action – Nabídne opravu chyby (např. import chybějící knihovny)
Mezerník + a + a	Otevře hlavní menu AI funkcí
Mezerník + a + c	Otevře chat s AI
Mezerník + a + e	Vysvětlí označený kód (Explain)
Mezerník + a + g	Vygeneruje kód podle zadání
Mezerník + a + m    Zobrazí výzvu ke změně modelu AI
```
